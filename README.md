# Pure CSS Playing Cards Layout

A responsive, semantic, and pure HTML/CSS implementation of classic playing cards. This project demonstrates advanced CSS layout techniques including Flexbox, CSS transforms, and precise positioning to replicate the detailed design of standard playing cards.

## Features

- **Pure HTML & CSS**: Built entirely with semantic markup and custom styling without relying on heavy frameworks or images.
- **Flexbox Architecture**: Utilizes Flexible Box Layout to handle card alignment, orientation, and responsive wrapping across devices.
- **HTML Entities**: Leverages clean HTML suit symbols (`&spades;`, `&clubs;`, `&hearts;`, `&diams;`) for sharp rendering at any scale.
- **Accurate Details**: Includes inverted bottom-right indicators using CSS `transform: rotate(180deg)` for an authentic card deck feel.
- **Responsive Design**: Adapts fluently to various viewport sizes using flexible sizing, gaps, and wrapping.

## Included Cards

The current layout showcases a diverse set of standard playing cards:
1. Ace of Spades (A♠)
2. Two of Clubs (2♣)
3. Three of Hearts (3♥)
4. Four of Clubs (4♣)
5. Five of Diamonds (5♦)

##How to Run the Project Locally
To view the project on your machine, follow these steps:

1. Clone the repository:  
```git clone "https://github.com/telesphore8527/Pure-Css-playing-Card"```
2. Navigate to the project folder:  
```cd pure-css-playing-cards```
3. Open the project:
Simply double-click the index.html file to open it in any modern web browser, or use an extension like Live Server in Visual Studio Code.
